# Python Project - REDI School 2nd Semester 2025: Der Die Das Game

## Overview
A game to help improve the learning of German noun genders.

German language learners of all levels often struggle with maintaining consistent language learning progress. 
Noun gender is one challenging aspects of the German language. Exploring German, 
reveals that there are consistent rules to determine noun gender.
This is a simple python app to help users on learning noun genders in the format of a game.

## Usage Guide

On the main screen, you can choose from the following game types:

### Random Nouns
- Choose between 10 to 50 nouns selected from all German nouns
- Tests your general knowledge
- Typically difficult to achieve perfect scores

### Nouns by Suffix
- Certain suffixes typically determine the article for a noun
- Includes both rule-following nouns and exceptions
- Tests your specific knowledge of gender determined by suffixes

### 100 Most Used Nouns
- 10 random nouns from the 100 most used nouns
- Ideal for beginners

### 500 Most Used Nouns
- Same principle as "100 Most Used Nouns"
- Increased difficulty with larger noun pool

### 1000 Most Used Nouns
- Samples from 1000 most used nouns
- Intermediate level difficulty

### Survive Mode
- Random nouns from 90,000+ Wiktionary.org entries
- Goal: achieve the highest streak of correct answers
- Game ends after a single mistake
- Beyond C2 level - for experts and academics

## Other Features

### Hint button
- Provides probabilities indicating likely gender based on patterns
- **Note**: Some percentages can be misleading due to exceptions
- Use cautiously as a learning aid
